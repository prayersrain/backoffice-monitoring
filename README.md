# 📈 Backoffice Monitoring (PWA)

A Progressive Web App (PWA) designed for real-time back-office monitoring. This application allows administrators to oversee operations, track inventory (e.g., iPhone registry), and manage internal processes efficiently from any device.

## ✨ Features
- **Progressive Web App (PWA)**: Installable on mobile and desktop for native-like experience.
- **Inventory Tracking**: High-speed, search-based input system for inventory management.
- **Offline Capabilities**: Reliable performance even in low-network conditions.

## 💻 Tech Stack
- Frontend: Next.js, TypeScript
- PWA: next-pwa
